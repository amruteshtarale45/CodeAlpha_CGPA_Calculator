# CGPA Calculator

This repository contains a C++ program to calculate the Semester Grade Point Average (SGPA) and Cumulative Grade Point Average (CGPA) of a student.

## Features

- Accepts input for the number of semesters.
- Calculates SGPA for each semester based on course grades and credits.
- Computes overall CGPA across all semesters.
- Simple console-based interface.

## Language

- C++

## How to Run

1. Compile the program using a C++ compiler.
   Example:
   ```
   g++ calculateCGPA.cpp -o calculateCGPA
   ```
2. Run the executable and follow the prompts to enter grades and credits.
